# 📦 Sistema de Controle de Produtos (C#)

Este projeto foi desenvolvido como parte da avaliação **CP1** da disciplina de C#, com o objetivo de aplicar conceitos fundamentais da linguagem, como entrada e saída de dados, estruturas condicionais e programação orientada a objetos.

---

## 🚀 Funcionalidades

* Cadastro de 2 produtos
* Entrada de dados via console:

  * Nome do produto
  * Preço unitário
  * Quantidade
  * Desconto (%)
* Cálculo automático:

  * Custo total (preço × quantidade)
  * Custo final (com desconto aplicado)
* Comparação entre produtos:

  * Identifica qual possui o maior custo final

---

## 🛠️ Tecnologias Utilizadas

* C#
* .NET
* Console Application

---

## 📋 Como Executar o Projeto

### Pré-requisitos

* .NET SDK instalado (versão 6 ou superior)

### Passo a passo

```bash
# Clone o repositório
git clone https://github.com/RaphaelGomesMancera/cp1-produtos-csharp.git

# Acesse a pasta do projeto
cd cp1-produtos-csharp

# Execute o projeto
dotnet run
```

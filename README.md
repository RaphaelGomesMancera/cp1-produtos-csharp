# CP1 - Sistema de Controle de Produtos

Projeto desenvolvido em C# como parte da CP1.

## 📌 Funcionalidades
- Cadastro de 2 produtos
- Cálculo do custo total
- Aplicação de desconto para valores acima de R$5000
- Comparação entre produtos

## 🛠️ Tecnologias
- C#
- .NET

```bash
dotnet run
